## 样式特点总结

1. **标题边框**：H1 和 H2 有底部边框线
2. **配色方案**：使用 GitHub 官方配色
3. **代码高亮**：浅灰背景，深色文字
4. **表格设计**：表头背景，清晰边框
5. **引用样式**：左侧彩色边框
6. **链接颜色**：GitHub 蓝色 (#0969da)

所有样式都与 GitHub.com 的 Markdown 渲染保持一致！
